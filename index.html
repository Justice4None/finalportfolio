Tower's Wake

Tower's Wake is a browser-based Tower Defense game, made almost completely in JavaScript with no use of Flash or other in-browser game engines. Made by Dirk Stockton, Ghett Lee, Travis Rodowick, Rebecca Hernandez and William Dunn, Tower's Wake is currently in Alpha v0.1. We originally made Tower's Wake for our first class project at the UC Davis Coding Bootcamp, but we've all agreed to continue development of the game as time allows.
